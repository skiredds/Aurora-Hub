# v1.11.0.0 - The General Update

## 🟢 Added:
#### Main Tab:
* **Server Hopper** — Quickly switch to a different server.
#### World tab:
* **Infinite Prompt Range** — Grants global interaction reach across the entire map. Automatically saves and restores original distance values when toggled off.
* **Camera Occlusion** — Enables Invisicam mode, allowing the camera to pass through walls and solid objects without zooming into the character.
* **Fastmode (Extended)** — Now includes No Fog, disabled shadows, and global material simplification (Smooth Plastic) for maximum FPS.
* **Lag Switch (Keybind)** — Forces extreme network replication lag. Freezes the world around you and desyncs your character from the server.
#### Teleport tab:
* **Copy Coordinates** — Added a button that copies your exact position to the clipboard.
#### Player tab:
* **Anti-Sit** — Optimized event-based listener that blocks the Seated state without CPU overhead.
* **Background Mode** — Dynamically lowers FPS to 15 when the window is unfocused and restores it to your previous FPS limit when you return.
* **FPS Unlocker** — Added a slider to manually set the FPS cap up to 999.

## 🟡 Changed:

* **Optimization** — Migrated WalkSpeed and JumpPower to an event-based system (`GetPropertyChangedSignal`). This reduces CPU overhead to nearly zero.
* **Noclip** — Optimized collision scanning method by switching to `GetChildren`, significantly increasing FPS during active use.

## 🔴 Removed:

* **Legacy Loops** — Removed `RenderStepped` logic to prevent engine task scheduler congestion and physics conflicts.

## 🔵 Fixed:

* **Stuttering Fix (Limping)** — Completely resolved the "limping" animation bug and character stuttering caused by high-frequency value overwriting.
* **Stability** — Fixed movement instability and rubberbanding issues. Value restoration is now perfectly synced with character physics.
